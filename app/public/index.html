About

Given a sketch of a person, this application converts the sketch into realistic image of the faces of that person. Generating an image is challenging as it involves creating all the pixels of an image. This can be learnt if the parameters to the distribution of pixels in the image is known. GANs (Generative Adversarial Neural Network) can be used to generate images by making the model to learn the pixel distribution of images. Given an input image GAN tries to generate a latent space based on many image parameters such as mean, variance, correlation between pixels and so on. Different GAN models such as pix2pix, cycleGAN, BicycleGAN are tuned to generate image sample. This app can mainly be used in forensics.
